下拉选择器

## 何时使用

- 简单的下拉选择

## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| onSelect | 布尔变量 | Boolea | false |
| labelInValue | 为 `true` 时渲染的组件 | Element | - |
| keyName | 为 `false` 时渲染的组件 | Element | - |
| valueName | 为 `false` 时渲染的组件 | Element | - |
| placeholder | 为 `false` 时渲染的组件 | Element | - |
| data | 为 `false` 时渲染的组件 | Element | - |
